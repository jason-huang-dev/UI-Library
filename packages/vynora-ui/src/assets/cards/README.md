# Vynora Card Assets

Use this folder as the card asset design system.

- `suits`: SVG/vector suit and tile symbols.
- `frames`: SVG/vector rarity and cosmetic frames.
- `backs`: card-back patterns and future WebP/PNG back art.
- `textures`: future WebP/PNG paper, foil, and table textures.
- `effects`: reusable SVG/vector effects.
- `tiles`: future East Asian tile-style symbols.

Prefer composing cards from layers instead of exporting every card as a full image.
